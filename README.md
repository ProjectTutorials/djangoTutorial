# djangoTutorial

## To get started
```
pip install django
```

```
pip install pipenv
```

### In the folder - Create skeleton
```
pipenv install django
```

```
django-admin startproject djangoProject .
```

### Run

```ps
python manage.py runserver
```
It should look like this at [localhost:8000](localhost:8000)
